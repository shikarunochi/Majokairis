# Majokairis 動画データから画像と音声を抽出するツール
マジョカアイリス（microSD付バージョン）のmicroSDカード内にある動画データ（拡張子 savi）から、jpeg画像と wav データを抽出します。
（いまのところ、抽出できたりできなかったりします。）
python で書いてます。
>python extract.py A006.savi
のように対象ファイルを指定して使ってください。
